# Private Working Docs

Use this folder for local-only working notes, drafts, research scraps, and other documents that should never be pushed to the shared repository.

Rules:
- Everything in this folder stays untracked except this `README.md` and the local `.gitignore`.
- If a private document needs to live elsewhere in the repo temporarily, give it a `*.private.*` filename so Git ignores it.
- Do not store passwords, API keys, tokens, recovery codes, or other secrets here. Use proper secret storage instead.
