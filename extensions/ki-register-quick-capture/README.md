# KI-Register Quick Capture (Chrome Extension)

Diese Extension oeffnet Quick Capture direkt aus der Chrome-Toolbar und kann optional Coverage Assist lokal auf diesem Geraet aktivieren.

## Ordner

`extensions/ki-register-quick-capture`

## Installation (lokal)

1. In Chrome `chrome://extensions` oeffnen.
2. `Entwicklermodus` aktivieren.
3. `Entpackte Erweiterung laden` klicken.
4. Diesen Ordner auswaehlen: `extensions/ki-register-quick-capture`.

## Nutzung

- Klick auf das Toolbar-Icon:
  - Popup mit Quick Capture, Coverage Assist Toggle und Status fuer den aktuellen Tab
- Shortcut:
  - `Cmd+Shift+Y` (macOS)
  - `Ctrl+Shift+Y` (Windows/Linux)
- Rechtsklick auf einer Website:
  - `KI-Register Quick Capture`
- Standard-Ziel ist fest:
  - `https://kiregister.com/capture?source=chrome-extension`
- Wenn du gerade auf `kiregister.com`, `app.kiregister.com`, `www.kiregister.com`, `localhost` oder `127.0.0.1` bist, nutzt die Extension automatisch diese Origin.
- Das Toolbar-Icon passt sich automatisch an Dark/Light Browser-Theme an.
