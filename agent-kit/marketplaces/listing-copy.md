## Name

KI-Register Agent Kit

## Short description

Documentation-first CLI and skill pack for AI applications, workflows, and EU AI Act-ready operational records.

## Long description

KI-Register Agent Kit helps teams document AI applications, processes, and workflows while the work is happening, not weeks later. It combines a lightweight Node CLI with a portable skill bundle so Codex, Claude Code, OpenClaw, and similar agent systems can create consistent workflow folders with human-readable `README.md` files and machine-readable `manifest.json` records.

The package is built for reviewability:

- onboard once with your role and defaults
- capture documentation during coding or agent work
- ask only for missing required fields
- require final confirmation before each write by default
- validate outputs against a stable manifest contract

This makes it easier to maintain intended-purpose records, human oversight checkpoints, risk notes, controls, and evidence references in a form that can support internal governance and AI Act-oriented documentation programs.
